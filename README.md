# Memory Science for Agent Design

**記憶の科学とエージェント設計** — 認知科学 × AI 技術 × Agentic UX を横断して学ぶ学習アプリ。

🔗 **公開 URL**: https://ryosei-hatakeyama.github.io/memory-science-app/

---

## 概要

Claude などの AI エージェントを「道具」から「自律的なパートナー」に変えるために必要な、記憶の科学とエージェント設計の知識を体系的に学ぶコース。

認知科学の知見（符号化・保持・想起・忘却）を AI エージェントの設計に接続し、実際に動く記憶システムを設計できることを目標にしています。

---

## 学習コンテンツ

16 のモジュールで構成：

- **Phase 1（M01-M04）**: 記憶の科学基礎 — 符号化・保持・忘却・想起
- **Phase 2（M05-M08）**: AI エージェントの記憶設計 — コンテキスト管理・外部記憶・RAG
- **Phase 3（M09-M12）**: Agentic UX — ハーネス設計・オーケストレーション・マルチエージェント
- **Phase 4（M13-M16）**: 実装・応用 — 記憶評価・倫理・組織実践・統合設計

---

## 学習設計の特徴

このアプリ自体が記憶科学を活用して設計されています：

- **テスト効果（Testing Effect）**: 各モジュールに確認テストがあり、想起練習で定着を促進
- **精緻化質問（Elaborative Interrogation）**: 「自分の言葉で説明」で深い処理を促進
- **処理水準（Levels of Processing）**: 概念 → 内省 → 確認テストの順で処理を深化
